# qa-and-coordination
Add a task for QA and a task for Coordination

To use it, create a new bookmark in Chrome (by dragging ANY url to your bookmarks bar). Right-click on the bookmark, click on "Edit", and in the place of the URL place this code:

```
javascript:eval("(jQuery && jQuery.getScript('//cdn.jsdelivr.net/gh/nicokupferscayla/qa-and-coordination@latest/script.js?t=' + (+(new Date()))))");
```
